Tabela Ölçüm Prototipi

Kurulum:
1) Python 3.10+ kurulu olsun
2) Terminalde:
   pip install pillow opencv-python numpy

Çalıştırma:
   python tabela_olcum_app.py

Kullanım:
1. Fotoğraf Yükle
2. Kart Seç -> referans kartın 4 köşesini işaretle
3. Nesne Seç -> ölçmek istediğin alanın 4 köşesini işaretle
4. Alan Hesapla

Not:
- Kart ve nesne aynı düzlemde olmalı
- En iyi sonuç için fotoğraf üstten çekilmeli
