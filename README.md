# Real World Area Measurement (Computer Vision)

## 🇬🇧 English

This project is a computer vision application that calculates the real-world dimensions and area of objects using a reference object (credit card).

### 🚀 Features
- Real-time object measurement
- Reference-based scaling (credit card calibration)
- Graphical User Interface (GUI)
- Image-based measurement
- Live camera measurement support

### 🧠 How It Works
1. A reference object (credit card) is selected.
2. Pixel-to-centimeter ratio is calculated.
3. User selects object corners.
4. The system computes real-world dimensions and area.

### 🛠️ Technologies
- Python
- OpenCV
- NumPy
- Tkinter

### ⚙️ Installation
pip install -r requirements.txt

### ▶️ Usage
Run with Python:
python tabela_olcum_app.py

Or (Windows):
Calistir_Tabela_Olcum.bat

### 📌 Notes
- Accuracy depends on camera angle and distance
- Reference object must be in the same plane

## 🇹🇷 Türkçe

Bu proje, bir referans nesne (kredi kartı) kullanarak nesnelerin gerçek dünya boyutlarını ve alanını hesaplayan bir bilgisayarlı görü uygulamasıdır.

### 🚀 Özellikler
- Gerçek zamanlı nesne ölçümü
- Referans nesne ile ölçekleme (kredi kartı kalibrasyonu)
- Grafiksel kullanıcı arayüzü (GUI)
- Görüntü üzerinden ölçüm
- Kamera ile canlı ölçüm desteği

### 🧠 Nasıl Çalışır
1. Referans nesne (kredi kartı) seçilir.
2. Piksel/cm oranı hesaplanır.
3. Kullanıcı nesnenin köşelerini seçer.
4. Sistem gerçek boyut ve alanı hesaplar.

### 🛠️ Teknolojiler
- Python
- OpenCV
- NumPy
- Tkinter

### ⚙️ Kurulum
pip install -r requirements.txt

### ▶️ Çalıştırma
Python ile:
python tabela_olcum_app.py

Windows için:
Calistir_Tabela_Olcum.bat

### 📌 Notlar
- Doğruluk kamera açısına ve mesafeye bağlıdır
- Referans nesne ile ölçülen nesne aynı düzlemde olmalıdır
